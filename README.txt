few - advanced ed, few text editor.
Few is a text editor created for Unix-like systems, part of the FikusTools project. This text editor is an advanced ed editor, it makes it easier to work with text, many commands, etc. were also lightened, but this editor is still very similar to ed.


Install:

git clone https://github.com/fikuspi/few/
cd few
chmod 777 make.sh
sudo ./make.sh
